## Developing

Built with [SvelteKit](https://kit.svelte.dev/docs/).

```bash
pnpm run dev
```

## Building

To create a production version of your app:

```bash
pnpm run build
```

You can preview the production build with `pnpm run preview`.

## Deploy

Deploys to Netlify with the default adapter.
